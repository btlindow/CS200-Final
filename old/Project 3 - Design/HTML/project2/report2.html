Analysis Descriptions

The ChocAn Data System stores provider and member information as well as information about services to be provided. It allows providers to bill ChocAn for services provided. The ChocAn Data System atomically generates reports for members and providers. Summary reports can be made for managers. Data is stored on local hard drives and backed up in a remote provider directory as text files. The ChocAn Data System can run on computers that have a Java Virtual Machine, and internet capabilities with card reader. These systems will communicate with others via intranet locally and internet for accessing the ChocAn directory. Users can interact with the system using a command line interface.
